# Dao Factory by Crypto Thibault

Created with Create-react-app

### Dependencies:

- Chakra-ui
- Web3 Hooks
- Ethers Js

### Dapp structure

```
Dapp
├── DaoFactory
├── DaoList
├── Dao
│   ├── ContractsLayout
│   │   ├── Governance ── Proposal
│   │   ├── Management ── Employee
│   │   └── Treasury ── Charge
│   └── Roles
```

### Install Repository

```zsh
git clone https://github.com/CryptoThibault/dao-factory-front.git
yarn
```

### Run website on Localhost:

```zsh
yarn start
```
